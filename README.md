# Реализация упрощенного модели автомобиля на C#

## 1. Система позволяет
-создавать различные типы транспортных средств (легковые, грузовые, электромобили);
-управлять их состоянием (запуск, остановка, ускорение, заправка/зарядка);
-проводить техобслуживание;
-хранить автомобили в гаражах.

## 2. Требования
-.NET 8 или выше;
-среда разработки - Visual Studio.

## 3. Как запустить
1. Откройте Visual Studio.
2. Создайте новый проект типа «Консольное приложение» (Console App) на C#.
3. Скопируйте файлы SimpleCarModel.cs и Program.cs в папку проекта (обычно в ProjectName/ProjectName/).
4. Убедитесь, что в свойствах проекта указана нужная версия .NET (не ниже 8.0).
5. Соберите проект: в меню Build → Build Solution (или нажмите Ctrl+Shift+B).
6. Запустите проект: в меню Debug → Start Without Debugging (или нажмите Ctrl+F5).
В результате будет запущен исполняемый файл (*.exe) из папки bin/Debug/net8.0/.
